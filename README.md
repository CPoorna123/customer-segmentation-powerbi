# Customer Segmentation & Purchase Behavior Analysis

## 📊 Project Overview
This project analyzes retail sales data to understand customer purchasing behavior and identify high-value customers using **RFM (Recency, Frequency, Monetary) analysis**.
The insights are visualized through an interactive **Power BI dashboard**.

## 🛠 Tools & Technologies

* SQL (MySQL)
* Power BI
* Excel / CSV Dataset

## 📁 Dataset
Online Retail dataset containing transaction data such as:

* Invoice
* StockCode
* Product Description
* Quantity
* Invoice Date
* Price
* Customer ID
* Country

## 📊 Key Analysis Performed

### 1️⃣ Revenue by Country

Identified countries generating the highest revenue.

### 2️⃣ Customer Segmentation

Segmented customers into:

* VIP
* Loyal
* At Risk

Using **RFM analysis**.

### 3️⃣ Monthly Sales Trend

Analyzed sales performance over time.

### 4️⃣ Top Customers

Identified top 10 customers contributing the most revenue.

### 5️⃣ Top Products

Analyzed products with the highest sales volume.

## 💡 Key Insights

* The **United Kingdom contributes the majority of revenue**.
* A small percentage of **VIP customers generate most of the sales**.
* Some products consistently perform better than others.
* Sales trends reveal seasonal purchasing behavior.
